Prevention

This is all about avoiding the anti-patterns, and mistakes that lead to poorly structured code.

Do Not Pass or Return Null Values

Systemization

Exceptional is a mindset not a single act. Follow the best practives everywhere.

This is about setting yourself up for success.

Do not catch Exception

unless you are catching a more specific exception first.

Even then...reserve it for a high level general catch all.

string cmdTxt = "SELECT Count(*) as MyCnt FROM MyTable";

try {
  cmd.connection.Open();
